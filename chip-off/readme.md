# Chip-Off

![Chip-Off](./chip-off.p8.png)

## About

Chip-Off is a fast-paced, simplified poker showdown where pixelated clothes meet tactical folds. Outsmart your opponent to strip away their layers, unlocking vibrant new hair colors with every victory. Deal the hand, play your luck, and see if you can strip the deck!

## Controls

- Cursor keys
- x : select cards
- c : submit / start
