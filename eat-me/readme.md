# Eat Me

![Eat Me](./eat-me.p8.png)

## About

Built in 72 hours for Mini Jam 186, this minimalist arcade survival flips the bird to the food chain by making you the world's most ambitious snack. Forget surviving: you need to get devoured to evolve, proving that in this ecosystem, "you are what eats you".

## Controls

- Cursor keys
- 'x' or 'c' key to start
