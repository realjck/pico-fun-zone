# Mushroom Party

![Mushroom Party](./mushroom-party.p8.png)

## About

Cooked up in just 60 minutes for the 60-minute Game Jam #2, this game challenges your stomach and your reflexes. Your mission is to scarf down every mushroom that matches the requested size and treat the others like they’re radioactive. It’s a high-speed buffet where size definitely matters.

## Controls

- Cursor keys
- 'x' key to start
